# nw-tutorialist

Writes tested software tutorials using cognitive science principles (CLT, progressive disclosure, TTFHW). Use when creating a new tutorial, updating an existing one, or generating tutorial test scripts. Produces copy-paste-ready markdown with checkpoint validation.

**Wave:** Other
**Model:** inherit
**Max turns:** 50
**Tools:** Read, Write, Edit, Bash, Glob, Grep, Task
