# nw-workshopper

Use for workshop design. Guides IT trainers through a complete, facilitator-ready workshop plan in one continuous interactive session — from topic to export-ready package using TBR, Bloom's taxonomy, and gamification. Invoke with *workshop "Your Topic".

**Wave:** Other
**Model:** inherit
**Max turns:** 80
**Tools:** Read, Write, Edit, Glob, mcp__miro__create-board, mcp__miro__create-frame, mcp__miro__create-sticky-note-item, mcp__miro__create-card-item
